# 🎓 Academix
Academix is a student-focused task management app. It combines the simplicity of a to-do list with modern features like an AI study assistant and leaderboards for a more engaging academic experience. 🚀

## ✨ Features
- ✅ Add, edit, and manage tasks with deadlines
- 🎨 Clean and minimal UI for distraction-free productivity
- 🤖 AI Assistant to help with study planning and quick answers
- 🏆 Leaderboards to gamify productivity with friends or peers
- 📊 Task completion tracking and progress overview
